# 🏠 House Draw With Turtle Python

A simple Python Turtle Graphics project that draws a house using code.

Perfect for beginners who want to learn:
- Python basics
- Turtle graphics
- Drawing with code
- Coordinates and movement

---

# 📸 Preview

This project creates a simple house drawing using the built-in Python `turtle` module.

---

# ✨ Features

- Simple and beginner-friendly
- Made with Python Turtle
- Easy to edit and customize
- Lightweight project

---

# 🛠 Requirements

- Python 3.x
- Turtle module (included with Python)

---

# 🚀 Installation

Clone this repository:

```bash
git clone https://github.com/aksa2i1123-creator/house-draw-with-turtle-python.git
```

Go to the project folder:

```bash
cd house-draw-with-turtle-python
```

Run the program:

```bash
python main.py
```
